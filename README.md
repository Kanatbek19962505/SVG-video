# SVG-video
CMYK Honeycomb Video FX is a browser—based video player that superimposes the effect of professional printing rasterization (Halftone) on video in real time.
CMYK Honeycomb Video FX is a browser—based video player that superimposes the effect of professional printing rasterization (Halftone) on video in real time.
Instead of standard pixels, the image is formed from a dense hexagonal grid (honeycomb). On the fly, the algorithm splits each frame into four printing channels (Cyan, Magenta, Yellow, Black), calculates their intensity, and shifts the centers of the color layers relative to each other. Due to this, the layers are visually mixed correctly in multiply mode, creating a juicy, clear and seamless color image without white gaps and "dirty" shades.
